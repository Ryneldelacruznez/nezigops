Simple Portfolio Website (HTML + CSS)

Files
- index.html    (Home)
- about.html    (About)
- projects.html (Projects)
- contact.html  (Contact)
- style.css     (Shared styles)

How to open
1) Double-click index.html
2) Use the navbar to visit About / Projects / Contact

Customize
- Replace "Your Name" with your real name in all HTML files
- Update email/phone/social links in contact.html
- Replace project titles/descriptions in projects.html
